# Локация жилого района
 В этом проекте я создал небольшую локацию жилого района из домов, деревьев, фонарных столбов и другого.  
 Все объекты здесь - это префабы. Некоторые из них являются скачанными ассетами, а остальные созданы мной. Я решил 
 создать локацию в вечернее время, чтобы свет от фонарных столбов был виден лучше.
